# 页面 03：练习页

## 页面功能
- 执行当前文章在当前等级下的段落练习。
- 负责录音提交、评分反馈、段落达标判定与等级推进。
- 提供“录音专注模式”：开始录音后进入全屏遮罩，聚焦当前练习内容。
- 实现“强引导推进”核心规则：
  - 当前等级下所有段落达标后才可升级等级。
  - 升级顺序固定：`L1 -> L2 -> L3 -> L4`。

## 页面布局
- 头部仅保留操作入口（不展示”你当前在第几步/第几段”的顶部提示）：
  - 右上角轻入口：进度抽屉、我的页面入口
- 中部练习区（默认精简）：
  - 当前文章、当前等级、当前段落（放在中部信息卡，不放在顶部）
  - 目标阈值（`>= 85%`）
  - 文本/遮挡内容展示
  - 录音与提交主操作
  - 本次结果摘要（分数、是否达标）
- 可展开完整区：
  - token 级对齐结果
  - 错词/差异统计
  - 最近成绩趋势
  - 等级达标矩阵（含跳过段标记）
  - 同步状态细节
- 录音专注遮罩（录音中出现，顶层覆盖）：
  - 全屏遮罩层，遮挡练习页其他元素
  - 顶部区域：`关闭/取消`、`录音中`状态、`REC`指示
  - 中部区域：当前段落练习文本（专注阅读/跟读，日语文章显示注音）
  - 底部区域：主按钮 `结束录音`

## 交互方式
- 开始录音：
  - 点击 `开始录音` 后进入录音专注遮罩。
  - 遮罩层期间禁用底层页面交互，避免误操作。
- 结束录音：
  - 点击 `结束录音` 退出遮罩，回到练习页并进入识别/提交流程。
- 取消录音：
  - 点击左上角 `关闭/取消` 退出遮罩，本次录音不提交。
- 提交练习：
  - 完成一次录音后提交，返回分数和对齐结果。
  - 分数 `>= 85%` 则该段在当前等级记为达标。
- 跳过本段：
  - 允许进入下一段，但该段记为未完成。
  - 本等级结束前必须补齐达标，否则等级不完成。
- 手动切等级：
  - 允许升降级，但必须提示“将重置当前等级进度”。
  - 升到下一等级后，不允许回退到已完成等级。
- 保存策略：
  - 提交、跳过、切等级、切文章、离页时立即同步。
  - 同步失败不阻断用户操作，提示“未同步”，后台重试。
