# 页面 06：进度轻入口（抽屉/弹层）

## 页面功能
- 作为“进度信息”轻入口，仅嵌入练习页右上角。
- 提供不中断当前任务的进度可见性。
- 展示同步状态与必要恢复信息，不承担主流程编辑。
- 不承担页面顶部“当前位置提示”功能。

## 页面布局
- 触发入口：
  - 仅练习页右上角 `进度` 按钮
- 抽屉内容区：
  - 当前步骤（导入/练习/结果）
  - 当前文章上下文（文章、等级、段落）
  - 同步状态（已同步/未同步/重试中）
  - 最近保存时间
  - 手动重试同步按钮

## 交互方式
- 打开方式：
  - 在练习页点击右上角轻入口展开抽屉，不离开当前页面。
- 同步失败处理：
  - 显示“未同步”状态与重试入口。
  - 用户可继续操作，不被阻断。
- 关闭方式：
  - 点击遮罩或关闭按钮收起抽屉，返回当前页面状态。
