# 页面 01：开始页（历史用户）

## 页面功能
- 作为历史用户登录后的动态入口页。
- 展示最近一次练习上下文，帮助用户快速恢复流程。
- 提供两个主动作：`继续上次`、`新建文章`。
- 无历史进度用户不进入本页，直接进入“文章导入与全屏文本校对页”。

## 页面布局
- 头部仅保留基础操作：
  - 页面标题
  - `我的`入口
- 主卡片区域（核心信息集中在中部，不在顶部做位置提示）：
  - 最近文章标题
  - 当前等级（L1-L4）
  - 当前段落位置
  - 操作按钮：`继续上次`、`新建文章`
- 底部轻提示：
  - 恢复说明或上次更新时间

## 交互方式
- 页面进入时根据服务端进度判断是否显示本页。
- 点击 `继续上次`：跳转到该文章最近未完成节点。
- 点击 `新建文章`：进入“文章导入与全屏文本校对页”。
