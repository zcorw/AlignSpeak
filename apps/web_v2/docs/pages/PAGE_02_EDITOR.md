# 页面 02：文章导入与全屏文本校对页

## 页面功能
- 作为主流程的内容准备页，承接新建文章。
- 统一接收三类文本来源：
  - 粘贴板导入（主引导）
  - 文件导入（`txt/md`、图片 OCR）（主引导）
  - 手动输入（弱引导）
- 对导入内容进行全屏校对、分段确认、语种确认。
- 在进入练习前完成基础合法性校验。

## 页面布局
- 头部仅保留工具入口，不展示流程位置：
  - 导入方式入口（粘贴板、文件、手动输入）
  - 语种检测结果与手动改写控件
  - `我的`入口
- 初始态主体区域：
  - 仅展示三种输入入口与说明
  - 不显示大文本编辑框
- 编辑态顶层区域（触发后出现）：
  - 大文本编辑框（所有来源最终汇入此处）
  - 分段预览提示（按换行作为段落边界）
  - 以全屏顶层遮罩形式显示，并遮挡其他页面元素
- 底部操作区：
  - 主要操作：`开始练习`
  - 次级操作：重新识别/清空草稿（可选）
  - 校验反馈区域（错误/警告）

## 交互方式
- 导入动作：
  - 粘贴板读取成功后写入文本框并触发语种检测。
  - 文件导入后先解析文本，再写入文本框并触发语种检测。
  - 用户选择手动输入后直接进入编辑态。
  - 任一输入方式触发后，打开顶层全屏编辑框并遮挡其他元素。
  - 用户可继续在文本框内手动修订。
- 分段规则：
  - 以换行为段落边界，用户可通过编辑换行调整段落。
- 语种规则：
  - 自动检测后展示结果，允许用户手动修改最终语种。
- 点击 `开始练习` 前校验：
  - 文本非空
  - 段落数 `>= 1`
  - 语种已确定
  - 文本长度 `>= 20` 字符
  - 段落数 `<= 30`
- 校验通过后：
  - 创建文章上下文并进入“练习页”。
