# 页面 05：我的页

## 页面功能
- 提供全流程可进入的独立页面入口。
- 以历史文章管理为主，支持恢复任一文章进度。
- 提供最小账号管理能力（用户名展示、修改密码入口）。

## 页面布局
- 头部仅保留基础导航，不展示流程位置：
  - 页面标题（我的）
  - 返回主流程入口（不强制）
- 主视觉区域（核心）：
  - 历史文章列表（按最近练习时间倒序）
  - 每条展示：文章标题、当前等级、当前段、最近更新时间
  - 每条动作：`继续练习`
- 弱化账号区（次要）：
  - 用户名
  - 修改密码入口

## 交互方式
- 打开历史文章：
  - 点击某条文章 `继续练习`，加载该文章服务端进度并恢复到未完成步骤。
- 切换文章：
  - 当前文章未同步数据存在时提示可能丢失风险。
  - 新文章与旧文章进度相互独立。
- 账号操作：
  - 仅提供轻量入口，不干扰主流程。
